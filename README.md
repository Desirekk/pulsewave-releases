# Pulsewave — Releases

This repository hosts compiled builds and the live update manifest for
**Pulsewave**, a geometric PvP arena shooter currently in **closed alpha**.

The game's source code is private. Only release artefacts and the manifest
that the launcher polls live here.

## For testers

You don't interact with this repository directly. Run the **Pulsewave
Launcher**; it auto-updates from `manifest.json` and downloads the matching
build for your operating system.

If you don't have the launcher yet and you've been invited to the alpha,
contact **Desirekk** for the launcher binary.

## What's where

- `manifest.json` — points the launcher at the current game build per platform.
- GitHub Releases — actual zipped game builds, attached as Release assets.

## Roadmap

- **Closed alpha** (now) — invite only via launcher.
- **Steam Playtest** (~3-4 months) — public access through Steam.
- **Steam Early Access** (~9 months) — paid release, $9.99 plus free cosmetic
  unlocks, optional cosmetic packs and battle pass.
